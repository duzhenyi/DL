1. **背景介绍**

   我的第一个开源.NET Core 项目，功能很多，BUG肯定有，代码不是特别好，大佬勿喷！开源，随便搞哈！

   #### 软件架构

   框架：.NET Core3.1

   ORM：SqlSugar

   数据库：Sqlserver08R2+

   后端UI：Layui

   日志记录：log4+nlog

   缓存:内存缓存+redis缓存

   授权：基于cookie的JWT授权

   IOC:使用Autofac框架进行构造函数注入
